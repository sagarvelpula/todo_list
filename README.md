# To-Do Web Application (Flask)

A simple web-based To-Do application built using Python and Flask.
Users can add, manage, and track their daily tasks through a clean web interface.

## Features

* Add tasks
* View tasks
* Simple UI using HTML templates
* Backend logic using Flask

## Technologies Used

* Python (Flask)
* HTML (Jinja templates)
* Docker (for deployment)

## Live Demo

https://todo-list-sagar.onrender.com

## How to Run Locally

pip install flask
python app.py

Then open browser:

http://127.0.0.1:5000
